# Computer Science Portfolio Project

This repository contains my personal **computer science portfolio website**, developed as part of my B201 Computer Science Lab project at GISMA University of Applied Sciences. The portfolio showcases my technical skills, projects, academic background, and professional CV.

---

## 📌 Project Overview

The goal of this project is to build a professional online portfolio that demonstrates my abilities in:

- Web development (HTML, CSS, JavaScript)
- Version control using Git and GitHub
- Deployment using GitHub Pages
- Technical documentation using LaTeX

The project includes:
- A responsive portfolio website
- A structured GitHub repository
- A professionally formatted CV created using LaTeX
